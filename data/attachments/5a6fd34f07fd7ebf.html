Create a Set with Repetitions

The advanced interval functionality allows you to create workouts with multiple sets and repetitions. This feature was designed to allow you track as much or as little information about an interval workout that you want. All fields are not required and may or may not be used depending on the specifics of your workout. Below is a quick tutorial to help you get the most out of this powerful feature.

There are two types of sets you can create. Distance based sets (such as 5 x 800m) and time based sets (such as 4 x 5:00). If you are creating a distance based set you will need to fill out the following fields:

  • The number of repetitions this set will include 1
  • The distance of each repetition (800m, 2k, 1mi, etc.) 2

If you are creating a time based set, you will fill out the following fields:

  • The number of repetitions this set will include 1
  • The amount of time (duration) of each repetition 3

Once you have filled out the set information, click the down arrow 4 to expand each individual repetition.

Record the Repetitions

In the example above, 5 repetitions of 800m each were expanded. You will now be able to record time, distance and recovery for each repetition based on your workout. Below is a description of each repetition field:

1 If you have created a distance based set, then all of the distances will be equal such as in the example above. Conversely, if you created a time based set where you tracked how much distance you covered in a fix time interval, then each distance may differ in length.

2 The amount of time (duration) that it took you to cover the distance of the repetition. If you have a time based set (such as 4 x 5:00), then all of the times would be equal.

3 The amount of recovery (in time) that you took between each repetition. In the example above, 1:30 recovery was given between each 800m repetition.

4 The amount of recovery (in distance) that you took between each repetition. If the above workout required a 200m recovery interval instead of a timed interval, that would be recorded here.

5 Click the down arrow to copy all of the data from the current repetition down to the next repetition if the data you are recording for the two repetitions are equal. This is simply a time-saving function.

6 Click the delete icon to delete a repetition from the current set.

7 Click the plus icon to add an additional repetition to the current set. In the above example, clicking the icon once would change the workout to 6 x 800m.

Adding Additional Sets

To add an additional set to the workout, click the plus icon 1. Clicking this icon will display the Set Recovery/Time Distance Fields (shown below).

Once you have added an additional set, you can then record a set recovery in time and/or distance 1/2. For example, you might create a second set of 4 x 200m but you want to record that you took a 5:00 rest before starting your second set.

If you would like to delete a set entirely including any repetitions that you have recorded for that set, click the delete icon 3.

Create a Set without Repetitions

If you do not care about the individual repetitions and just want to record that you completed a set, you can do so by filling out the Reps, Distance and Duration fields shown below.

In the example above, type in the amount of repetitions 1 you completed, the distance of each repetition 2, and the average time it took you to complete each repetition 3. If you save the workout like the example above and not record individual repetitions, the training log will record that you completed 4000m of distance over a period of 15:00.
